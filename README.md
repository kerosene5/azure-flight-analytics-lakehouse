## Overview

Flight Analytics Lakehouse is an end-to-end Azure Data Engineering project implementing the Medallion Architecture (Bronze, Silver, and Gold layers) for large-scale flight operations data. The platform ingests raw flight datasets into Azure Data Lake Storage Gen2, applies SQL-based transformations and data quality processes, and delivers analytics-ready datasets for business intelligence and visualization.

The project showcases modern data engineering practices including cloud storage architecture, ETL/ELT design, data modeling, transformation workflows, and dashboard development using Microsoft Azure technologies.
